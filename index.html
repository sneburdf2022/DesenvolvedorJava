Aula inicial de html


1 - Ter o Git instalado no seu computador.

2 - Ter uma conta no github.

3 - Configurar o git no seu computador (configuração só é feita 1 vez)

3.1 - git config --global user.name "w3schools-test"

3.2 - git config --global user.email "test@3schools.com"

4 - iniciar o git em uma pasta/projeto (só é feito 1 vez)

4.1 - git init

4.2 - ou pode ser feito pelo vccode esta na gravação do dia 03.10

5 - fazer o commit das alterações no projeto (sempre fazer quando for criar um sabe point

5.1 - git add *

5.1.1 - git commit -m "mensagem obrigatoria do sabe point"

6 = enviar o seu código para seu repositório no github

6.1 - git remote add origin https://seuendereconogithub/nomeprojeto/nomearquivo.git

6.2 - ou pode ser feito pelo vscode esta na gravação do dia 03.10

7 - atualizar o projeto local com o código fonte guardado no github

7.1 - git pull origin
